### 🩺 Smart AI Healthcare Companion with RAG

Smart AI Healthcare Companion with RAG is an AI-powered healthcare assistant built using Streamlit, LangChain, and Large Language Models (LLMs). The system provides intelligent, safe, and context-aware wellness guidance, symptom analysis, and healthcare support using Retrieval-Augmented Generation (RAG).

---

## ✨ Features

* 🤖 AI-powered healthcare chatbot
* 🩺 Symptom-based wellness guidance
* 🥗 Diet and nutrition recommendations
* 🏃 Exercise and fitness suggestions
* 😴 Sleep and stress management support
* 📚 Context-aware responses using RAG
* ⚠️ Safe and non-diagnostic healthcare assistance
* 🌐 Modern Streamlit interactive UI
* 🔍 Personalized health-related guidance

---
